Video Source

User Interface > Control Settings

Settings that provide the video source for a control.

Properties

Property Description
IPCamera An IPCamera that serves as a fixed video source for a control.
IPCameraSelector A table of variable values and their associated cameras.
IPCameraSelectorVariable A Variable that selects the control's video source from a set of cameras in the IPCameraSelector table.
