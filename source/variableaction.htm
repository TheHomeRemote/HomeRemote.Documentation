Variable Action

Devices > Variables

Represents an action in the Actions collection of a VirtualDeviceVariable.

Properties

Property Description
Name The identifying name of the object.
ActionValue The value that is passed to the ActionVariable when the VirtualDeviceVariable value matches the Value property of this item.
ActionVariable The Variable that is set when the VirtualDeviceVariable value matches the Value property of this item.
Value The VirtualDeviceVariable value that triggers the action.

Remarks

Multiple actions can exist with the same Value. This means that you can update several device Variables when a single VirtualDeviceVariable is set.
