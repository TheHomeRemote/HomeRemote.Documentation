Voice Command

Voice Commands

Represents a voice command.

Properties

Property Description
Name The identifying name of the object.
ListenFor The phrase that triggers the command. (See Remarks)
Feedback The message reported when the app begins to execute the command.
Success The message sent when the voice command completes successfully.
Failure The message sent when the voice command fails.
CommandVariable The variable that is set when the command executes.
CommandValue The value the variable is set to when the commmand executes.

Remarks

All voice commands you give your mobile device must be preceded by the Command Prefix. The default value for this is "Home Remote". So if your ListenFor is "Turn on kitchen lights", the command you need to speak is "Home Remote, Turn on kitchen lights". The Command Prefix can be changed but it must be a valid string. It cannot be empty.
