Media Item Layout

User Interface > Control Settings

Settings that define the layout for items in a Media Browser control.

Properties

Property Description
HorizontalSpacing The horizontal spacing between columns.
ImageMargin The width of the left, top, right, and bottom margins of the item's Image.
ItemAlignment A TextAlignment value that sets the horizontal alignment of item content.
ItemHeight The height of the layout area for each item that is contained in a MediaBrowser.
ItemWidth The width of the layout area for each item that is contained in a MediaBrowser.
TitleMargin The width of the left, top, right, and bottom margins of the item's Title.
VerticalSpacing The vertical spacing between rows.
