System Requirements

Introduction

The Designer

  • The designer requires a PC with Windows 7 or later along with the .Net Framework 4.6.1

The Apps

  • The Android app requires Lollipop version 5.0 or later
  • The iOS app requires iOS 7.0 or later
  • The Windows Phone app requires Windows Phone 8.1 or later
  • The Windows desktop and tablet app require Windows 8.1 or later
