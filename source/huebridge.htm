Hue Bridge

Devices > Sources

Represents a connection to a Philips Hue bridge.

Properties

Property Description
HostName The host name or IP address of the bridge.
Name The identifying name of the object.
RefreshRate The rate in milliseconds that the connection should refresh its content.
UserName The account used to authorize connection with the bridge.

Remarks

The UserName is automatically set after you add the device to your project. This an identifier assigned by the bridge after you click the link button on the product. You should not change this.
