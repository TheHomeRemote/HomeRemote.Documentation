KNX Client

Devices > Sources

Represents a connection to either a KNX gateway or router.

Properties

Property Description
ConnectMethod A value that specifies whether the client uses a Routing or Tunneling connection.
GatewayAddress The gateway address used for a Tunneling connection.
GatewayPort The gateway port used for a Tunneling connection.
MulticastAddress The multicast address used for a Routing connection.
MulticastPort The multicast port used for a Routing connection.
Name The identifying name of the object.
