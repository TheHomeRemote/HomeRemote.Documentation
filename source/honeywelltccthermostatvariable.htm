Honeywell TCC Thermostat Variable

Devices > Variables

Represents a variable that is linked to an attribute in a Honeywell Total Connect Comfort Thermostat.

Properties

Property Description
Attribute The thermostat attribute that the variable is associated with.
Name The identifying name of the object.

Attributes

Attribute Access Description
cool_setpoint Read/write The desired temperature when cooling.
fan_mode Read/write The fan mode (Auto, On, Circulate).
heat_setpoint Read/write The desired temperature when heating.
hvac_mode Read/write The HVAC system heating/cooling modes (Heat, Cool, Off).
indoor_humidity Read-only The humidity level measured at the device.
indoor_temperature Read-only The temperature measured at the device.
outdoor_humidity Read-only The humidity level outside the dwelling.
outdoor_temperature Read-only The temperature outside the dwelling.
