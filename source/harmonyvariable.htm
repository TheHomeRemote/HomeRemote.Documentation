Harmony Variable

Devices > Variables

Represents a variable that is linked to an activity or device in a HarmonyHub connection.

Properties

Property Description
Category A value that specifies whether the variable is tied to an activity or device.
Command The device IR command that the variable is associated with.
Identifier The identifying name of the activity or device.
Name The identifying name of the object.
