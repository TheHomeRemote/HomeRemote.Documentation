UPnP Belkin Light Switch 1

Devices > Sources

Represents a connection to a UPnP device of type urn:Belkin:device:lightswitch:1.

Properties

Property Description
Name The identifying name of the object.
RootUDN The UDN for the root UPnP device.
UDN The UDN for the UPnP device.

Remarks

Please use the UPnP Belkin Light Switch 1 dialog to set your RootUDN and UDN. When this dialog opens it searches your network for devices of this type. Once your device is found, select it and click OK. This will automatically set or update your RootUDN and UDN values. A demonstration of this device being used can be found in our UPnP Example.
