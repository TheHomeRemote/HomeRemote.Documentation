HTTP Client

Devices > Sources

Represents a generic HTTP connection.

Properties

Property Description
Headers A collection of HTTP request headers. (Optional)
Name The identifying name of the object.
RefreshRate The rate in milliseconds that the connection should should send a StatusRequest. (Optional)
StatusRequest An optional HTTP request that is sent to update variable display values. (See Remarks)
URI The base address of the Internet resource used when sending requests.

Remarks

The StatusRequest will only be sent if one of its child HttpVariables has the StatusSource property set to StatusRequest. If a StatusRequest is provided it should return either JSON or XML data. This request is directly linked to the StatusPath property of a HttpVariable which must be either a JSONPath or XPath expression. That expression is used to point to a specific value in the response data. A demonstration of this device being used can be found in our HTTP Client Examples.
