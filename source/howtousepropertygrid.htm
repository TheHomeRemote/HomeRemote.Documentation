How to: Use Property Grid

Getting Started

Many properties on a control can be customized. By default all controls have a transparent background. By default, all controls have a font size 12px. Use the steps outlined below to customize a property

To customize a property:

  1. Click the button to right of the property you wish to customize.
  2. Select the "Customize" option.
  3. Update the property.
