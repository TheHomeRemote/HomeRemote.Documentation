Combo Box

User Interface > Controls

A control that presents a drop-down list of items a user can select from.

Properties

Property Description
Appearance The TextAppearance settings of the control.
ApplyForegroundToImage A value that determines whether the item's foreground is applied to the image.
Height The suggested height of the control.
ItemAppearance The TextAppearance settings of all unselected items.
Items A collection of ListItems displayed in the control.
ImageMargin The width of the left, top, right, and bottom margins of the item's Image.
ItemAlignment A TextAlignment value that sets the horizontal alignment of item content.
ItemHeight The height of the layout area for each item that is contained in a GridView.
Left The distance between the left side of the control and the left side of its parent Canvas.
Name The identifying name of the object.
SelectedItemAppearance The TextAppearance settings of the selected item.
SelectedValueVariable A Variable bound to the SelectedValue of the selection control.
TextMargin The width of the left, top, right, and bottom margins of the item's Text.
Top The distance between the top of the control and the top of its parent Canvas.
Visibility The Visibility settings of the control.
Width The suggested width of the control.
ZIndex The ZIndex determines the order on the z-plane in which the control appears.
