Image Appearance

User Interface > Control Settings

Settings that define the appearance for a control that display's an image.

Properties

Property Description
Background The ThemeColor applied to the background of a control.
Foreground The ThemeColor applied to the opaque parts of an image.

Remarks

Using the Foreground property for image controls is not encouraged for projects that will be deployed to Windows tablets and desktops. Image's that have the Foreground property set may appear a little distorted in our Windows Store desktop and tablet apps.
