KNX Variable

Devices > Variables

Represents a variable that is linked to group addresses in a KnxClient.

Properties

Property Description
DatapointType The DPT ID that defines the scaling and formatting of the variable value.
Name The identifying name of the object.
ReadAddress The group address that serves as the source for the variable's display value.
WriteAddress The group address that serves as the destination for variable write operations.

Remarks

All properties are optional:

If the DatapointType property is blank, the variable will not be scaled and it will default to hexadecimal formatting.
If the ReadAddress property is blank, the variable will not provide status. It will always display an empty string.
If the WriteAddress property is blank, the variable will ignore write operations.
