MyQ Gateway

Devices > Sources

Represents a connection to a MyQ Internet Gateway from either Chamberlain or LiftMaster.

Properties

Property Description
Name The identifying name of the object.
Password The password used to authorize connection with the gateway.
RefreshRate The rate in milliseconds that the connection should refresh its content.
UserName The user name used to authorize connection with the gateway.
