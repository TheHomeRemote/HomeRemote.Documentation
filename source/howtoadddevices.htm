Getting Started

Adding Devices and Variables

Use the Devices tab in the Project pane to add your devices. After a device or variable is added it can be updated in using the Properties pane. Additionally, double-clicking an existing device will display the same user-friendly dialog used when the device was added. You can also update devices with this dialog.
