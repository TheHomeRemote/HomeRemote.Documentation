Wink Variable

Devices > Variables

Represents a variable that is linked to a device attribute in a Wink Hub.

Properties

Property Description
Attribute The attribute that the variable is associated with.
ControlType The control type for the variable. When a variable is set should it update a Device or activate a Scene.
DeviceID The unique system identifier for a Wink device.
DeviceType The type of Wink device.
Name The identifying name of the object.
SceneID The unique system identifier for a Wink scene (or shortcut).
UseDesiredStateAsStatus Display the desired state for the monitored attribute, not the actual state.
