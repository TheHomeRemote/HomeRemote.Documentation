Represents a connection to a Vera controller.

Properties

Property Description
Name The identifying name of the object.
Password The password used to authorize connection with the controller.
SerialNumber The serial number of the controller.
UserName The user name used to authorize connection with the controller.
Version The version of firmware installed on the controller (UI5, UI7).
