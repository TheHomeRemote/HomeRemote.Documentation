How to: Use Themes

Getting Started

Background

  • The use of light and dark themes is supported on all devices exluding Windows 8.1 desktops and tablets. These devices will always use a light background.
  • All Windows Phone devices will use the background theme as defined in your phone's settings. Windows 10 devices will use the color mode value found on the Settings screen within the app itself. Other devices will use the value that is defined in the project.

Accent Color

  • All devices support the use of accent colors.
  • All Windows Phone 8.1 and Windows 10 devices will use the accent color of the device. Other devices will use the value that is defined in the project.
