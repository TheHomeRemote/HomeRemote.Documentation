Web Image

User Interface > Controls

Represents a control that displays an image from a web source.

Properties

Property Description
Height The suggested height of the control.
Left The distance between the left side of the control and the left side of its parent Canvas.
Name The identifying name of the object.
RefreshRate The rate in milliseconds that the control should refresh its content.
RefreshTrigger A Trigger that reloads the image from the control's URL.
Top The distance between the top of the control and the top of its parent Canvas.
Url The URL that provides the path to an image to display in the control.
Visibility The Visibility settings of the control.
Width The suggested width of the control.
ZIndex The ZIndex determines the order on the z-plane in which the control appears.

Remarks

The control will refresh its content when the control is first loaded and whenever the Refresh button is clicked. Automatic refreshing can be accomplished by providing either a RefreshTrigger or a RefreshRate greater than -1.

This control can also be used to display video from an IP camera by setting the Url property to the snapshot URL provided by the camera manufacturer. You will also need to set the RefreshRate to 0 or a value greater than 0.

Credentials for the connection can be defined in the URL like so:
http://username:password@IPADDRESS:80/snapshot.cgi
