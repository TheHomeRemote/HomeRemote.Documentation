Button Type

User Interface > Control Settings

Defines the behavior of a button control and how it interacts with a control's state variable.

Members

Member Description
Normal Specifies that the variable should be set after a button is pressed and released.
Momentary Specifies that the variable should be set when a button is pressed and reset when a button is released.
Pulse Specifies that the variable should be set after a button is pressed and released. After 250 ms the variable is then reset.
Toggle Specifies that, while in Off state, the variable should be set after a button is pressed and released. While in On state, the variable should be reset after a button is pressed and released.

Remarks

There is only one command associated with a variable. This command is executed each time the variable is set and reset. The difference between the two is that when a set is performed the OnState value is passed in as an argument. When a reset is performed the OffState value is passed in as an argument.

See OnOffStates.
