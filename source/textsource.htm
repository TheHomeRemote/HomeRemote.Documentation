Text Source

User Interface > Control Settings

Settings that provide text for a control.

Properties

Property Description
Text A string that specifies the static text of a control.
TextSelector A table of variable values and their associated strings.
TextSelectorVariable A Variable that selects the control's text from a set of strings in the TextSelector table.
TextVariable A Variable whose value is bound directly to the text of a control.
