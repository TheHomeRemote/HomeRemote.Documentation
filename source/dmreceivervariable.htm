DM Receiver Variable

Devices > Variables

Represents a variable that is linked to a status and/or command in a DM receiver.

Properties

Property Description
Attribute The receiver status and/or command that the variable is associated with.
Name The identifying name of the object.

Attributes

Attribute Access Description
hdradio_artist Status The artist playing on the HD Radio tuner.
hdradio_band Status/command The AM/FM band of the HD Radio tuner. Values include AM and FM.
hdradio_channel_down Command Used to send a channel down command to the HD Radio tuner.
hdradio_channel_up Command Used to send a channel up command to the HD Radio tuner.
hdradio_frequency Status/command The frequency of the HD Radio tuner expressed in MHz.
hdradio_mode Status/command The mode of the HD Radio tuner. Values include MANUAL, AUTO, AUTOHD, ANAMANU, ANAAUTO.
hdradio_multicastchannel Status The HD channel of the current FM frequency.
hdradio_programtype Status The program type of the broadcasting HD radio station.
hdradio_stationname Status The name of the station playing on the HD Radio tuner.
hdradio_title Status The song title playing on the HD Radio tuner.
hdradio_signallevel Status The signal strength of the current HD channel.
hdradio_album Status The album playing on the HD Radio tuner.
hdradio_genre Status The genre of the station playing on the HD Radio tuner.
network_browser_back Command Navigate back to the previous level of items in the network browser, if there is one.
network_browser_isopen Status A value of true indicates that the network browser is open and a value of false indicates that the network browser is closed.
network_browser_line1_text Status The text for the first line in the network browser.
network_browser_line1_type Status The line type for the first line in the network browser. Values range from 0 to 3.
network_browser_line2_text Status The text for the second line in the network browser.
network_browser_line2_type Status The line type for the second line in the network browser. Values range from 0 to 3.
network_browser_line3_text Status The text for the third line in the network browser.
network_browser_line3_type Status The line type for the third line in the network browser. Values range from 0 to 3.
network_browser_line4_text Status The text for the fourth line in the network browser.
network_browser_line4_type Status The line type for the fourth line in the network browser. Values range from 0 to 3.
network_browser_line5_text Status The text for the fifth line in the network browser.
network_browser_line5_type Status The line type for the fifth line in the network browser. Values range from 0 to 3.
network_browser_line6_text Status The text for the sixth line in the network browser.
network_browser_line6_type Status The line type for the sixth line in the network browser. Values range from 0 to 3.
network_browser_line7_text Status The text for the seventh line in the network browser.
network_browser_line7_type Status The line type for the seventh line in the network browser. Values range from 0 to 3.
network_browser_linerequest Command Open the contents of a provided line number ranging between 1 and 7.
network_browser_next Command Navigate to the next page of items in the network browser.
network_browser_open Command Opens the network browser. After 15 seconds of inactivity the browser is closed.
network_browser_status Status The status of the browser that indicates the current page number and total number of pages.
network_browser_previous Command Navigate back to the previous page of items in the network browser.
network_player_album Status The name of the album playing on the receiver's network audio source.
network_player_artist Status The artist playing on the receiver's network audio source.
network_player_albumart Status A value of true indicates that the song playing on the network player has album art available and a value of false indicates no album art is available. (See remarks)
network_player_albumart_changed Status A false to true trigger that occurs when the album art for the network player changes.
network_player_bitrate Status The bitrate of the song playing on the receiver's network audio source.
network_player_next Command Play the next network audio track.
network_player_playback Status/command The current Play/Pause state of the network player.
network_player_previous Command Play the previous network audio track.
network_player_source Status The audio source playing in the nework player.
network_player_track Status The name of the artist playing on the receiver's network audio source.
network_source Status The name of the current network audio source.
network_title Status A short description representing the content of either the network browser or player.
zone1_mute Status/command Mute control for zone 1. Values include OFF and ON.
zone1_power Status/command Power control for zone 1. Values include OFF and ON.
zone1_source Status/command The current source for zone 1. Values vary between models (see Remarks).
zone1_volume Status/command The volume for zone 1. Values range from -80 to 18 (zero decimal places).
zone2_mute Status/command Mute control for zone 2. Values include OFF and ON.

zone3_mute Status/command Mute control for zone 3. Values include OFF and ON.
zone3_power Status/command Power control for zone 3. Values include OFF and ON.
zone3_source Status/command The current source for zone 3. Values vary between models (see Remarks).
zone3_volume Status/command The volume for zone 3. Values range from -80 to 18 (zero decimal places).

Remarks

Album art for the receiver's network audio can be displayed in a WebImage control. The URL for the album art is http://<hostname>/NetAudio/art.asp-jpg.



Source values for Marantz SR7007 include M-XPORT, HDRADIO, CD, TV, PANDORA, FAVORITES, IRADIO, SIRIUSXM, SERVER, FLICKR, SPOTIFY.
