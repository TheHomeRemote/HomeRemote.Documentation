MyQ Variable

Devices > Variables

Represents a variable that is linked to a device attribute in a MyQ Gateway.

Properties

Property Description
Attribute The device attribute that the variable is associated with.
DeviceID The ID of the device to control. (See remarks)
Name The identifying name of the object.

Attributes

Attribute Access Description
desired_door_state Write-only The desired state for a garage door opener (0=Close, 1=Open).
door_state Read-only The current state for a garage door opener (1=Opened, 2=Closed, 4=Opening, 5=Closing).
light_state Read/write The state for a remote light control (0=Off, 1=On).

Remarks

The MyQ variable dialog will automatically populate the DeviceID field. Please invoke the dialog to find your devices.
