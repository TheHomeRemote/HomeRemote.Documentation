Hue Variable

Devices > Variables

Represents a variable that is linked to a light, group, or scene in a HueBridge.

Properties

Property Description
Attribute The bridge attribute that the variable is associated with.
Category A value that specifies whether the variable is tied to a light, group, or scene in a Hue bridge.
Identifier The identifying name of the light, group, or scene in a Hue bridge.
Name The identifying name of the object.
