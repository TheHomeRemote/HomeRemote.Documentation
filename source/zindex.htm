ZIndex

User Interface > Control Settings

Command that adjusts the order on the z-plane in which a control appears.

Commands

Command Description
Bring to Front Brings the control to the front of the z-order.
Bring Forward Brings the control forward one position in the z-order.
Send Backward Brings the control back one position in the z-order.
Send to Back Sends the control to the back of the z-order.
