How to: Add Controls

Getting Started

Use the following method to add controls:

Steps

  1. Click the control you wish to add in the ribbon.
  2. Move the mouse over to the design surface and select an area. This will set the size and location of the control.
  3. Configure the control's properties.

Remarks

At the root of every page should be a Canvas control. By default a Canvas control is included for all new pages. If this gets deleted a new one should be added in its place.
