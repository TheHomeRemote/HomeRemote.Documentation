Controls

User Interface

Topics in this section:

Control Description
Button Represents a button control that reacts to touch events.
Canvas Defines an area within which you can explicitly position child elements by using coordinates that are relative to the Canvas area.
ColorWheel A control that presents a wheel of colors a user can select from.
ComboBox A control that presents a drop-down list of items a user can select from.
DigitalClock Represents a control that displays current date and/or time.
GridView A control that presents a collection of items in rows and columns.
HyperlinkButton Represents a button control that functions as a hyperlink.
Image Represents a control that displays an image.
ImageButton Represents a button control with an image (instead of text) that reacts to touch events.
Label Represents a control that displays text.
Page Represents a page of content that can be navigated to.
PageBrowser Represents a control that hosts and navigates pages.
PageGroup Represents a group of pages.
MediaBrowser A control that presents a collection of items in rows and columns.
MediaElement Represents an object that renders audio and video to the display.
MjpegView Represents a control that displays a MJPEG video stream.
RadioButton Represents a button that allows a user to select a single option from a group of options.
Rectangle Represents a control used to draw a solid colored rectangle.
Slider Represents a control that lets the user select from a range of values by moving a thumb along a track.
TextBox Represents a control that can edit a single line of text.
ToggleSwitch Represents a switch that can be toggled between two states.
WebBrowser Represents a control that hosts and navigates HTML content.
WebImage Represents a control that displays an image from a web source.
