SmartThings Variable

Devices > Variables

Represents a variable linked to a command action and/or attribute in a SmartThings Hub.

Properties

Property Description
Attribute The attribute that provides state information for the variable. (Optional)
Capability The capability that the variable is associated with.
DeviceID The unique system identifier for a SmartThings device.
Name The identifying name of the object.

Remarks

Please note that the Attribute property is only required if you wish to provide feedback for a variable. Not all commands provide feedback and not all commands are directly linked to an Attribute, so there will be cases when the Attribute should be left blank. In these situations the command that is sent will be whatever the OnState or OffState is for a Button control, for example. For more information about capabilities and attributes please refer to the SmartThings Documentation.
