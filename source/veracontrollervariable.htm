Vera Controller Variable

Devices > Variables

Represents a variable linked to a command action and/or display variable in a Vera Controller.

Properties

Property Description
DeviceID The ID of the vera device to control.
CommandService The service that sends an action to the vera device.
CommandAction The action the vera device should perform.
DisplayService The service used to retrieve the status of a vera variable.
DisplayVariable The name of the vera variable.
Name The identifying name of the object.

Remarks

The dialog used to add a Vera variable in the HomeRemote designer will automatically configure these properties. It will display the user friendly names of your Vera devices as well as its available controls. This eliminates the need for you to know the actual device ID's, service names, etc. Once added, each of these individual properties can be customized in our Property Grid.
