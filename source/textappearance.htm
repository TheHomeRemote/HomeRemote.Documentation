Text Appearance

User Interface > Control Settings

Settings that define the appearance for a control that displays text.

Properties

Property Description
Background The ThemeColor applied to the background of a control.
FontSize The font size for the text content.
FontStyle A value that specifies the style of the text as either Normal or Italic.
FontWeight A value that specifies the thickness of the text as either Normal or Bold.
Foreground The ThemeColor applied to the text contents of a control.
