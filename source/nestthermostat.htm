Nest Thermostat

Devices > Sources

Represents a connection to a Nest thermostat.

Properties

Property Description
Name The identifying name of the object.
SerialNumber The serial number of the thermostat.
Structure The ID of the structure containing the thermostat.
UserName The user name used to authorize connection with the thermostat.
Password The password used to authorize connection with the thermostat.

Remarks

The dialog used to add or configure a thermostat connection includes a search feature to find your Nest thermostats. This will automatically populate the SerialNumber and Structure fields.
