Page Group

User Interface > Controls

Represents an item in the app's navigation pane that contains a group of pages.

Properties

Property Description
ImageName The image shown with the page group in the navigation panel.
Name The identifying name of the object.

Remarks

The ImageName property is only applicable for the Android and Windows apps. The Windows Phone app does not display images in the navigation panel.
