Image Source

User Interface > Control Settings

Settings that provide an image for a control.

Properties

Property Description
ImageName A string that specifies the static image of a control.
ImageSelector A table of variable values and their associated imagese.
ImageSelectorVariable A Variable that selects the control's image from a set of images in the ImageSelector table.
