Insteon Variable

Devices > Variables

Represents a variable that is linked to a device or scene in an InsteonHub connection.

Properties

Property Description
Attribute The hub attribute that the variable is associated with.
Category A value that specifies whether the variable is tied to a device or scene.
Identifier The identifying name of the device or scene.
Name The identifying name of the object.
