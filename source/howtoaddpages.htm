How to: Add Pages

Getting Started

In order to begin adding pages you must first create page groups. Page groups are the items shown in the app's navigation pane. All pages must reside within a page group.

Adding Page Groups

  1. Click either the PageGroup button in the ribbon or the "New" button in the Project pane.
  2. Configure the Name and ImageName property of the PageGroup in the Properties pane.

Adding Pages

  1. Like with page groups, click either the Page button in the ribbon or the "New" button in the Project pane. All new pages get added to the page group that is currently selected.
  2. Configure the Name property of the Page in the Properties pane

Remarks

Right-clicking the app design surface toggles the visibility of the navigation pane.
