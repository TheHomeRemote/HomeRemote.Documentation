Crestron Controller Variable

Devices > Variables

Represents a variable that is linked to join in a Crestron® Controller.

Properties

Property Description
JoinType A value that specifies whether the variable is tied to an analog, digital, or serial join in a SIMPL Windows® program.
JoinNumber A value that specifies the join number of the corresponding signal in a SIMPL Windows® program.
Name The identifying name of the object.

Remarks

A variable of a digital join type will either have a value of "false" or "true". When assigning one of those variables to a button, the control's OffState should be set to "false" and OnState set to "true". Please note, that the default ButtonType for a button is Normal. This means that clicking a button will only set the digital join to "true", it won't reset it. For behavior that mimics that of the Crestron Mobile® app you should also change the ButtonType to Momentary. One disadvantage of the Momentary ButtonType is that it can be prone to accidental clicks when performing, say, a screen swipe with your fingers. An alternative to the Momentary ButtonType is a Pulse. This will set and then reset the join after an actual button click. The duration of the pulse is 250 ms.

See Button, ImageButton, ButtonType, and OnOffStates.
