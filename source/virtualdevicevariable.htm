Virtual Device Variable

Devices > Variables

Represents a variable defined in a Virtual Device.

Properties

Property Description
Actions A set of actions to execute when the variable is set. (See VariableAction)
DefaultValue A value that specifies the default value for the variable when the app is first started.
MappingVariable A Variable that is linked to the VirtualDeviceVariable through a set of mappings.
VariableMappings A collection of mapping variable values and their associated Variables.
Name The identifying name of the object.

Remarks

The use of mappings is optional. They allow for the creation of complex variables built on a combination of actual device variables.
