Control Settings

User Interface

Topics in this section:

Control Setting Description
ButtonType Defines the behavior of a button control and how it interacts with a control's state variable.
ImageAppearance Settings that define the appearance for a control that display's an image.
ImageSource Settings that provide an image for a control.
Keyboard Represents a virtual keyboard on a mobile device.
MediaItemLayout Settings that define the layout for objects in a Media Browser control.
MediaObjectImage Represents an image for a media object in a Media Browser control.
OnOffStates Settings used to define a control's On/Off states.
ShapeAppearance Settings that define the appearance for a shape control.
TextAlignment Specifies whether text is centered, left-aligned, or right-aligned.
TextAppearance Settings that define the appearance for a control that display's text.
TextSource Settings that provide text for a control.
ThemeColor Represents the color for a control based on the project's theme.
Trigger Settings used to trigger a control request from a variable.
VideoSource Settings that provide the video source for a control.
Visibility Settings that link a variable to a control's visibility.
ZIndex Represents a command that adjusts the order on the z-plane in which the control appears.
