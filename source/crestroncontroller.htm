Crestron Controller

Devices > Sources

Represents a connection to a Crestron® control system.

Properties

Property Description
HostName The host name or IP address of the control system.
Name The identifying name of the object.
Port The TCP/IP port number used for communication with the control system.
Passcode The passcode used to authorize connection with the control system.

Remarks

The Crestron Mobile® symbol to communicate with your control system. The Port and Passcode properties should match those of the Crestron Mobile® symbol in your SIMPL Windows® project.
