How to: Create a Project

Getting Started

Before creating a new project, first decide whether you want to start from scratch or modify a template. Templates can provide a good way to get you started.

Creating a Project from Scratch

  1. Click File->New.
  2. Set the project's orientation (Portrait/Landscape)
  3. Set the design platform, screen size, and aspect ratio (see Remarks).
  4. Begin adding your Devices and Pages.

Modifying a Template

  1. Download a template from http://thehomeremote.com/.
  2. Open the file.
  3. Set the simulator platform, screen size, and aspect ratio to match your target device.
  4. Configure your devices and variables. Devices can be configured from either the Properties pane or the Device dialog. The dialog is bit more user friendly and includes search features for Nest thermostats and Vera controllers. It can be opened by double-clicking the device item in the Project pane.
  5. Delete unused content.

Remarks

You can change the Design settings at anytime for a project however be aware that doing so could require changes to all your page layouts. For example the design area on a 10" Windows tablet is greater than on a 10" Android tablet. The design area is less on Android platforms in order to accomodate the action bar, tab items, and status bars. Windows tablets don't have these items. This means that if you change the design platform or screen size you will likely have to update your page layouts.
