UPnP Media Renderer 1 Variable

Devices > Variables

Represents a variable that is linked to an attribute in a UPnP Media Renderer 1.

Properties

Property Description
Attribute The attribute that the variable is associated with.
Name The identifying name of the object.

Attributes

Attribute Access Description
current_play_mode Read/write The play mode for the device (normal, shuffle, repeat_one, repeat_all, random, direct_1, intro).
current_track Read-only The track number.
current_track_album Read-only The album playing on the device.
current_track_album_art_uri Read-only The uri to the album's artwork.
current_track_creator Read-only The artist playing on the device.
current_track_duration Read-only The duration for the current track.
current_track_title Read-only The song title playing on the device.
mute Read/write Mute control for the device (0, 1).
next Write-only Play the next track.
number_of_tracks Read-only The number of tracks in the current playlist.
pause Write-only Pause the current track.
play Write-only Play the current track.
previous Write-only Play the previous track.
record Write-only Record the current track.
relative_time_position Read-only The time position for the current track.
stop Write-only Stop the current track.
transport_state Read-only The state of the player (stopped, playing, transitioning, paused_playback, paused_recording, recording, no_media_present, unknown).
volume Read/write Volume control for the player (0-100).
