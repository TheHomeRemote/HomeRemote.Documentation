Media Object Image

User Interface > Control Settings

Represents an image for a media object in a Media Browser control.

Properties

Property Description
ImageName A name that specifies the image of an object.
Name The identifying name of the object.
Class The class for the media object. (See Remarks)

Remarks

These definitions are used to determine the image that is displayed for objects in a Media Browser control. If the server associated with the control provides artwork for an object that will be displayed, however not every object has artwork. The image that is used for those objects will be based on the object's class type. Every media object has an assigned class type. Examples include container class types such as object.container.storageFolder and object.container.playlistContainer as well as item classes such as object.item.audioItem.musicTrack and object.item.videoItem.movie. So if you want a particular image to be displayed for objects of class type object.container.storageFolder that have no artwork, choose an image and set the class type to object.container.storageFolder. Because it's hard to foresee every possible class type we recommend adding an image to this collection with the Class property left blank. That image will be considered the default for the Media Browser control. It will be used for all objects that have no server defined artwork and no specific class entry in the collection.
