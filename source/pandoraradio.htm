Pandora Radio

Devices > Sources

Represents a connection to the Pandora Internet Radio music service.

Properties

Property Description
Name The identifying name of the object.
Password The password used to authorize connection with the music service.
UserName The user name used to authorize connection with the music service.

Remarks

You can only play your stations to a Sonos UPnP Media Renderer. You can browse your stations with the Media Browser control. A demonstration of this music service being used can be found in our Sonos Example.
