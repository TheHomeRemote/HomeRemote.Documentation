UPnP Dimmable Light 1 Variable

Devices > Variables

Represents a variable that is linked to an attribute in a UPnP Dimmable Light 1.

Properties

Property Description
Attribute The attribute that the variable is associated with.
Name The identifying name of the object.

Attributes

Attribute Access Description
level Read/write The dimming level of the light (0-100).
power Read/write The powered state of the light (0, 1).

Remarks

A demonstration of this device being used can be found in our UPnP Example.
