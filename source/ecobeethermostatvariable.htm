ecobee Thermostat Variable

Devices > Variables

Represents a variable that is linked to an attribute in an ecobee Thermostat.

Properties

Property Description
Converison The conversion that should be applied to the attribute value (None, Fahrenheit, Celsius).
Name The identifying name of the object.
Path The path to the attribute that the variable is associated with.
ThermostatID The ID of the thermostat to control.

Remarks

All temperatures default to an integer value of Fahrenheit times 10. To use the actual Fahrenheit units for a variable please choose a Conversion of Fahrenheit as it will divide the raw value by 10. To use the actual Celsius units choose a Conversion of Celsius.
