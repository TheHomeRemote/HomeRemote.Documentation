Nest Thermostat Variable

Devices > Variables

Represents a variable that is linked to an attribute in a Nest Thermostat.

Properties

Property Description
Attribute The thermostat attribute that the variable is associated with.
Name The identifying name of the object.

Attributes

Attribute Access Description
ambient_temperature_c Read-only The temperature measured at the device in half degrees Celsius.
ambient_temperature_f Read-only The temperature measured at the device in whole degrees Fahrenheit.
away Read/write The away state of the thermostat (home, away).
hvac_mode Read/write The HVAC system heating/cooling modes (heat, cool, range, off).
target_temperature_c Read/write The desired temperature in half degrees Celsius.
target_temperature_f Read/write The desired temperature in whole degrees Fahrenheit.

Remarks

The target and ambient temperatures are unitless. To set the temperature to 70°F a value of 70 should be written to the variable.
