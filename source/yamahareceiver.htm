Yamaha Receiver

Devices > Sources

Represents a connection to a Yamaha AV receiver.

Properties

Property Description
HostName The host name or IP address of the AV receiver.
Name The identifying name of the object.
RefreshRate The rate in milliseconds that the connection should refresh its content.

Remarks

Click here to download an example project configured for a Yamaha AV receiver. Just double-click the Yamaha AV Receiver item on the DEVICES tab and provide your receiver's host name or IP address. Click OK and you are all set.
