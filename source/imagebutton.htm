Image Button

User Interface > Controls

Represents a button control with an image (instead of text) that reacts to touch events.

Properties

Property Description
Appearance The ImageAppearance settings used while in an undefined state.
ButtonType The ButtonType value that determines when the state variable is set during touch events.
Height The suggested height of the control.
Increment The value by which the button's state variable is increased or decreased
Left The distance between the left side of the control and the left side of its parent Canvas.
Name The identifying name of the object.
OffAppearance The ImageAppearance settings used while in Off state.
OffImage The Image that should be displayed while in Off state.
States The OnOffStates of the control.
OnAppearance The ImageAppearance settings used while in On state.
OnImage The Image that should be displayed while in On state.
Image The Image that should be displayed while in an undefined state.
Top The distance between the top of the control and the top of its parent Canvas.
Visibility The Visibility settings of the control.
Width The suggested width of the control.
ZIndex The ZIndex determines the order on the z-plane in which the control appears.
